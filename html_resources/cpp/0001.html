怎样自学c++呢？

我前段时间正好在学习 C++ 基础知识，分享下我的学习过程吧。

从入门到放弃

很多新人一上来就学那本超级厚《C++ Primer Plus》（我刚开始也是），然后过程如下：

1.C++ 那么牛逼吗？初学者一来就要学这个砖头书，难不倒我！（亢奋）

看玩前言、导论，大神们的介绍果然牛逼，看起来也不难嘛，我誓必然的学好！（毒誓）

开始进入章节 1 个小时后... 挖槽？我刚看了什么，我睡着了？还睡了 40 分钟？ （放弃）

不夸张，这就是我当时学习的心理路程 ...

就这么放弃吗？

不，我是一个不会放弃的男人，既然看书不适合我（初学者），我去看视频学习！！！

没错，我入门 C++ 是靠看视频，单纯看视频就看了吗？ 并不是

在看视频的同时，会边做笔记 + 边写代码实操。这很重要，不要以为视频里的代码例子简单，就不动手实操了，人类对自己最大的错觉就是以为自己懂了。

只有动手实操了，你才会发现原来自己刚才那么多细节没看懂（哈哈哈哈）....

B 站众所周知是个很好的「学习网站」，推荐下当初我看的视频：视频链接

推荐的理由是：每节课都是一个小知识点，每节课的视频长度不长，并且每一阶段会有个小项目例子教学，对初学者是比较友好的，推荐 1.5 倍播放速度哦！

特此申明：我不是黑马培训班的学员！！！

回归原始

就只看视频就行了吗？

答案当然不是，看视频的好处是先初学者掌握 C++ 重要（常用）的知识点，那具体更多的细节当然还要回归到书籍上面。

所以在学习 C++ 的时候，最好的姿势是： 看完视频一个章节，然后回归《C++ Primer Plus》看对应的章节，这样看起书来会比较轻松！

总之就是这么个过程：

看完一个章节视频 -> 实操视频例子 -> 看对应的章节的书

以上就是我在入门 C++ 的姿势，希望对你有帮助。
